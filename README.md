# Character Counter

A simple and responsive Character Counter web app that lets users track the number of characters and words as they type — built with HTML, CSS, and JavaScript. Deployed live on GitHub Pages.

🔗 **Live Demo:** [gopinath-33.github.io/Character-Counter](https://gopinath-33.github.io/Character-Counter/)

## 📋 Overview

This project is a lightweight text utility tool that counts characters and words in real time as the user types or pastes content into a text box. It's a great mini-project to practice DOM manipulation and event handling in JavaScript.

## ✨ Features

- Real-time character count as you type
- Real-time word count
- Clean, minimal, and responsive UI
- Instant feedback with no page reload (pure JavaScript event listeners)
- Simple, distraction-free interface

## 🛠️ Tech Stack

- **HTML5** – Structure and layout
- **CSS3** – Styling and responsive design
- **JavaScript** – Logic for counting characters/words in real time

## 📁 Project Structure

```
Character-Counter/
├── index.html      # Main HTML structure
├── style.css       # Styling for the app
├── script.js       # Character/word counting logic
└── README.md       # Project documentation
```

## 🚀 Getting Started

To run this project locally:

1. Clone the repository
   ```
   git clone https://github.com/Gopinath-33/Character-Counter.git
   ```
2. Navigate to the project folder
   ```
   cd Character-Counter
   ```
3. Open `index.html` in your browser
